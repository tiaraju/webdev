Nosso objetivo será estudar tecnologias web "modernas". Embora as tecnologias fundamentais tenham sido desenvolvidas há algo entre 20 e 25 anos, diversas melhorias e inovações para a web são continuamente propostas. Além disso, não há como restringir-se a uma única tecnologia que atenda às peculiaridades das diversas partes e/ou etapas do desenvolvimento de uma aplicação web. Desenvolver para a web, portanto, requer que se compreenda e domine várias tecnologias, incluindo protocolos, linguagens, ferramentas, opções de design e de arquitetura, boas práticas, etc. Fica evidente, portanto, que é impossível que um curso cubra de forma exaustiva as tecnologias ou que ao final de um curso um estudante as domine completamente. É por isso que defino os objetivos gerais deste curso de forma modesta: introduzir o estudante aos conceitos e tecnologias fundamentais da web.
