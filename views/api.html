API da Biblioteca Prog3 - UFCG

Book

Recurso referente a um livro.

GET /book/

Retorna um JSON contendo o status da requisição e uma lista de todos os livros salvos no BD dentro do objeto results.

POST /book/

Adiciona um novo livro na biblioteca. O corpo da mensagem deve estar em formato JSON. Os seguintes campos são necessários:

  • title: Título do livro.
  • writers: Lista de autores do livro. Deve ser passado uma lista de Strings(i.e: ["paul","john",...,]
  • description: Descrição do livro.
  • price: Preço do livro.
  • cover: Url da capa do livro.
  • comments:Lista de comentários sobre o livro

*Title, writers e price são obrigatórios

PUT /book/:bookId

Substitui um livro já existente. Espera-se um JSON que descreva um livro no corpo da mensagem.
